## ‚öôÔ∏è Installation
To get a local copy up and running, follow these simple steps.

### Prerequisites
-   Node.js (LTS version recommended)
-   npm (comes with Node.js)
-   MongoDB (Community Edition recommended)
-   Git

### Setup
1. Clone the repo
   ```bash
   git clone https://github.com/aditya6100/SalesLens.git
   ```
2. Navigate to the project directory
   ```bash
   cd SalesLens
   ```
3. Install frontend dependencies
   ```bash
   cd client # Assuming your React app is in a 'client' folder
   npm install
   cd ..
   ```
4. Install backend dependencies
   ```bash
   cd server # Assuming your Node.js app is in a 'server' folder
   npm install
   cd ..
   ```
5. Configure environment variables
   ```
   # Create a .env file in the 'server' directory and add necessary variables (e.g., MongoDB connection string, API keys)
   # Example:
   # MONGO_URI=mongodb://localhost:27017/saleslens_db
   # JWT_SECRET=your_jwt_secret
   ```

## üöÄ Usage
### Start the Backend
```bash
cd server
npm start
```
### Start the Frontend
```bash
cd client
npm start
```
After starting both, open your browser to `http://localhost:3000` (or whatever port your frontend runs on).
